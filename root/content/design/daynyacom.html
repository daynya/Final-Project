Ongoing development of personal website

Ever-evolving development of my personal portfolio site. In lieu of social media, I would like for this to take on the role of a constantly updated personal journal. I'd like to showcase my work, both on and off the computer, as well as give you some insight into what I am up to. The current version uses Bootstrap for the grid, Signal, as well as a little JavaScript.

June, 2019

The most recent development of my site involves a few new details. There is JavaScript put in place to swap the images based on the screen width. There are changes to the homepage upon screen resizing. I added a "What's new" page that I plan to use as a journal.

Early 2019

I took things in a whole new direction - brighter, more clean and better organized. I used a blocky grid layout to achieve a fun new look, with a fully responsive portfolio grid finally put in place.
